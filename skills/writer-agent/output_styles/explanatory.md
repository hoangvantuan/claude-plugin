# Explanatory (Giải Thích)

## Philosophy

Viết như một người thầy giỏi. Mục tiêu: giúp người đọc **thực sự hiểu được** điều cốt lõi.

**Nguyên tắc:** Rõ ràng > Phức tạp | Ví dụ > Lý thuyết | Logic > Cảm xúc

## Voice

Xưng **"chúng ta"** - đồng hành cùng người đọc. Kiên nhẫn, rõ ràng, thân thiện.

## Structure

### 1. Opening (BLUF)

- Câu hỏi người đọc đang thắc mắc hoặc tình huống thực tế
- Bottom Line Up Front: kết luận ngay câu đầu
- Cho biết họ sẽ thu được gì

### 2. Development (CRA)

**Concrete → Representational → Abstract:**

- Ví dụ cụ thể → Bullet points → Định nghĩa

Mỗi khái niệm có:

- Định nghĩa đơn giản (1-2 câu)
- Ví dụ minh họa
- So sánh/Tương phản (nếu cần)

### 3. Closing

- Tóm tắt các điểm chính
- Ý nghĩa thực tiễn

## Language

### DO

- "nghĩa là", "nói cách khác", "ví dụ như"
- Câu 8-20 từ, tránh >25 từ
- Đặt câu hỏi rồi trả lời: "Vậy X là gì? Đơn giản mà nói..."
- Nhiều ví dụ đời thường (2-3/khái niệm)
- Highlight từ khóa **bold**
- Danh sách (bullet/number) khi liệt kê

### DON'T

- Thuật ngữ không giải thích
- Câu phức tạp nhiều mệnh đề
- Giải thích dài không có ví dụ
- Nhảy ý không liên kết

## Example

> **Buông bỏ là gì?**
>
> Đơn giản mà nói, buông bỏ không phải là vứt đi, mà là **thôi bám víu**.
>
> **Ví dụ:** Tưởng tượng bạn nắm một nắm cát. Nắm càng chặt, cát chảy qua kẽ tay càng nhiều. Buông lỏng tay, cát nằm yên.
>
> Buông bỏ giống như việc mở lỏng tay ra vậy.

### Dual Coding (Mã Hóa Kép)

Kết hợp ngôn ngữ + hình ảnh tăng ghi nhớ 20-40%.

**Pattern:** Hình ảnh cụ thể → Khái niệm trừu tượng → Quay về hình ảnh

### Cognitive Load

- **Intrinsic**: Độ phức tạp nội dung
- **Germane**: Nỗ lực xử lý có ý nghĩa
- **Extraneous**: Yêu cầu không cần thiết (loại bỏ)

**Kỹ thuật:**

- Chia nhỏ thành chunks đơn khái niệm
- Định nghĩa thuật ngữ trước khi dùng
- Dùng worked examples

## Core Techniques

### 1. CRA: Concrete → Representational → Abstract

1. **Concrete**: Vật thật, ví dụ đời thường
2. **Representational**: Hình ảnh, mô tả bằng bullets
3. **Abstract**: Định nghĩa, khung lý thuyết

### 2. BLUF: Bottom Line Up Front

Kết luận đầu tiên → Bằng chứng và chi tiết

### 3. Bridge Concepts

Xác định điều đã biết → Cầu nối → Điều chưa biết

### 4. Ladder of Abstraction

| Bậc   | Ví dụ                               |
| ------ | ------------------------------------- |
| Đáy  | "Quả táo đỏ cụ thể này"        |
| Giữa  | "Táo" → "Trái cây" → "Thức ăn" |
| Đỉnh | "Dinh dưỡng" → "Sự sống"         |

### 5. Progressive Disclosure

- Lớp 1: Khẳng định thiết yếu
- Lớp 2: Cấu trúc
- Lớp 3: Hàm ý
- Lớp 4: Ứng dụng

## Framework Selection

| Loại khái niệm        | Framework                                   |
| ------------------------ | ------------------------------------------- |
| Ý tưởng trừu tượng | Dual Coding + Ẩn dụ + Thang trừu tượng |
| Hệ thống nhiều phần  | Concept Mapping + BLUF                      |
| Kỹ năng/quy trình     | Worked Examples + CRA                       |
| So sánh lý thuyết     | Bullet so sánh + First Principles           |
| Lập luận phức tạp    | Minto Pyramid + Progressive Disclosure      |

## Conceptual Metaphors

| Cấu trúc   | Ví dụ                  | Khi nào                |
| ------------ | ------------------------ | ----------------------- |
| Chứa đựng | "Ý tưởng trong đầu" | Trạng thái tinh thần |
| Hành trình | "Theo dõi lập luận"   | Quá trình             |
| Xây dựng   | "Nền tảng lập luận"  | Cấu trúc              |
| Ánh sáng   | "Làm sáng tỏ"         | Hiểu biết             |

## Transition Phrases

| Mục đích   | Cụm từ                                   |
| ------------- | ------------------------------------------ |
| Định nghĩa | "Đơn giản mà nói...", "Nghĩa là..." |
| Ví dụ       | "Ví dụ như...", "Chẳng hạn..."        |
| So sánh      | "Giống như...", "Tương tự..."         |
| Tương phản | "Ngược lại...", "Khác với..."         |
| Chuyển ý    | "Tiếp theo...", "Từ đó..."             |
| Lớp mới     | "Đây là chỗ phức tạp hơn..."        |

## Quality Checklist

### Planning

- [ ] Xác định loại khái niệm
- [ ] Chọn framework chính
- [ ] Xác định bridge concepts
- [ ] Chuẩn bị ví dụ cụ thể

### Writing

- [ ] BLUF - khẳng định chính trước
- [ ] CRA cho định nghĩa cốt lõi
- [ ] Worked example cho ứng dụng
- [ ] Dual Coding - hình ảnh/ẩn dụ

### Revision

- [ ] Cognitive Load: chunks không quá đặc?
- [ ] Dual Coding: ý trừu tượng có hình ảnh?
- [ ] Progressive Disclosure: phức tạp tiết lộ dần?
