# Overview Article Template

## Purpose

The overview article (00-overview\.md) is the single entry point for the series.
It combines introduction and key takeaways.

## Structure

### 1. Hook (1-2 sentences)

Capture attention with:

* Surprising fact

* Provocative question

* Relatable problem

### 2. Topic Introduction (1 paragraph)

* Define the subject clearly

* Provide context/background

* Establish scope

### 3. Why It Matters (1 paragraph)

* Relevance to reader

* Real-world impact

* Current importance

### 4. What You'll Learn (bullet list)

* 3-5 key outcomes

* Concrete skills or knowledge

* Promise of value

### 5. Key Takeaways (numbered list)

* Core concepts from the series

* 3-7 main insights

* Each with brief explanation

### 6. Series List (at the end)

* All articles with links and descriptions

* Current article highlighted

* Same format as other articles

## Tone Guidelines

* Welcoming, not academic

* Confident, not arrogant

* Clear, not oversimplified

* Engaging, not sensational

## Length

Target: 400-600 words

## Two-Phase Creation

### Phase 1 (Step 4.1)

Create initial overview with sections 1-4.
Use placeholders for Key Takeaways and Series List:

```markdown
## Điểm chính
_[Sẽ cập nhật sau khi hoàn thành tất cả bài viết]_

## Các bài viết trong series
_[Sẽ cập nhật sau khi hoàn thành tất cả bài viết]_
```

### Phase 2 (Step 5)

Update overview with actual Key Takeaways and Series List.

## Template

```markdown
# [Series Title]

[Hook sentence that captures attention]

## Giới thiệu

[Topic introduction paragraph]

## Vì sao quan trọng

[Significance paragraph]

## Bạn sẽ học được gì

- [Outcome 1]
- [Outcome 2]
- [Outcome 3]

## Điểm chính

1. **[Concept 1]**: [Brief explanation]
2. **[Concept 2]**: [Brief explanation]
3. **[Concept 3]**: [Brief explanation]

## Các bài viết trong series

1. **Tổng quan - Brief description** _(đang xem)_
2. [Article 1 Title](./01-slug.md) - Brief description
3. [Article 2 Title](./02-slug.md) - Brief description
4. [Article 3 Title](./03-slug.md) - Brief description
```

## Quality Checklist

* [ ] Hook is compelling

* [ ] Topic is clearly defined

* [ ] Value proposition is clear

* [ ] Key takeaways capture core insights

* [ ] Series list is complete with working links

* [ ] Tone matches chosen style

* [ ] Within word count range (400-600 words)
